Blackhole

Explanation

Blackhole is the codename for the current infrastructure at dn0.eu, hosting Matrix servers, Gitlab instances, S3 storage spaces and websites (such as jae.fi or the very website your are viewing right now).

This page is just as a description of that infrastructure.

Routers

NODEMASTER (FRA)

This machine is a BGP router running Debian and hosted in Germany by iFog Gmbh.

It also has WireGuard running on it to provide connectivity to my personal computers and phones to internal services.

Nodes

All nodes marked with HEL are hosted in Helsinki at an undisclosed location.

ANANKE (HEL)

This is the machine that runs mostly static websites, blogs, pads and VaultWarden.

PHOBOS (HEL)

This machine is handling all Matrix related traffic from servers to media repos. It is one of the beefiest machines of the infrastructure (behind TRITON).

PANDORA (HEL)

This is the big storage box, having a MinIO server running on it and handling all the storage for most applications.

TRITON (HEL)

This machine is handling everything related to hosted GitLab instances, it is the most beefy machine of the infrastructure.

CHARON (HEL)

This machine is handling traffic related to emails.
